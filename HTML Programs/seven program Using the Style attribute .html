Introduction to HTML

HTML(Hyper Text Markup Language)is standard markup language for creating wab pages.It defines the structure and layout of a wab document by using a variety of tags and atteributes.

Each HTML tag serves a specific purpose such as a defining a headings,paragraphs,links,images,form and more.for example,the &It;h1>to&It;h6>tag define headings of different levels of importance,while the &It;p>tag defines the paragraphes of text.

HTML documents are structured as a hierarcy of elements nested within each other.The&It;html>tag represent the root of documents,contaning two main section:the&It;head>section of meta information and the &It;body>section for the visible content of the page.

Here's baisc example of an HTML document:

		
			 <!DOCTYPE html>
            <html>
            <head>
                <meta charset="UTF-8">
                <title>My First HTML Page</title>
            </head>
            <body>
                <h1>Welcome to My Website</h1>
                <p>This is a paragraph of text.</p>
                <a href="https://www.example.com" style="color: #0066cc; text-decoration: none;">Visit Example.com</a>
            </body>
            </html>

HTML is a foundation technology for wab development and is comlemented by CSS(Cascading Style Sheets)for styling and JavaScript for dynamic behavior.
